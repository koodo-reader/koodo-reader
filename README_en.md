## Develop

Make sure node's version on your computer is larger than 10.0.0, yarn and git is in place.

1. Download the repo

   ```
   git clone https://github.com/troyeguo/koodo-reader.git
   ```

2. Enter desktop version

   ```
   yarn
   yarn dev
   ```

3. Enter web version

   ```
   yarn
   yarn start
   ```
